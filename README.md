# Инструкция по запуску CRM "СОКОЛ"

## 1. Как открыть терминал
1. Нажмите `Command + Space` (Spotlight).
2. Введите `Terminal` и нажмите Enter.

## 2. Как запустить программу
Скопируйте и вставьте следующие команды в терминал:

### Вариант А (через готовый скрипт):
```bash
cd /Users/angrycat/.gemini/antigravity/scratch/client-manager
bash run.sh
```

### Вариант Б (вручную):
```bash
cd /Users/angrycat/.gemini/antigravity/scratch/client-manager
streamlit run app.py --server.port 8502
```

## 3. Как открыть в браузере
После запуска терминал покажет ссылки. Обычно это:
http://localhost:8502

## 4. Как остановить
Чтобы выключить программу, вернитесь в терминал и нажмите `Ctrl + C`.
Просто закрыть окно терминала тоже сработает (программа остановится).

---
**Почему "порт не открывается"?**
"Порт" — это как дверь, в которую стучится браузер. Эту дверь держит открытой сама программа. 
Если программа не запущена (вы выключили компьютер, закрыли терминал или произошла ошибка), дверь закрывается, и браузер пишет "Не удается получить доступ".
