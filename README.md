# Client Manager

Система управления клиентами для ипотечных брокеров.

## 🚀 Деплой на Streamlit Community Cloud

Это приложение готово к бесплатному размещению в облаке Streamlit.

### Инструкция

1.  **Загрузите код на GitHub**
    - Создайте новый репозиторий (можно приватный).
    - Загрузите файлы проекта.

2.  **Подключите Streamlit Cloud**
    - Зайдите на [share.streamlit.io](https://share.streamlit.io/).
    - Нажмите "New app".
    - Выберите свой репозиторий, ветку (обычно `main`) и главный файл: `app_new.py`.

3.  **Настройте секреты (Secrets)**
    - В настройках приложения на Streamlit Cloud откройте раздел "Advanced settings".
    - В поле "Secrets" скопируйте содержимое вашего файла `.env` в таком формате:

    ```toml
    SUPABASE_URL = "ваша_ссылка_на_supabase"
    SUPABASE_KEY = "ваш_ключ_supabase"
    YANDEX_TOKEN = "ваш_токен_яндекса"
    ```
    *(Кавычки обязательны!)*

4.  **Запустите (Deploy!)**
    - Нажмите кнопку "Deploy".

### ✨ Готово!
Ваше приложение будет доступно по ссылке вида `https://client-manager-app...streamlit.app`.
